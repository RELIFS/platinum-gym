# Dependency Package Reference

Dokumentasi dependency utama tersedia pada:

[docs/dependency.md](docs/dependency.md)

File `docs/dependency.md` berisi:

- Identitas dokumentasi dependency.
- Ringkasan status penggunaan package.
- Daftar dependency/package proyek.
- Dependency backend terpasang.
- Dependency development dan testing.
- Dependency frontend.
- Dependency rencana pengembangan.
- Analisis 5W+1H tiap dependency utama.
- Cara install dependency.
- Analisis perubahan `composer.json`, `composer.lock`, `package.json`, dan `package-lock.json`.
- Dampak dependency pada proyek.
- Risiko umum dependency.
- Refleksi kelompok.

File ini dipertahankan sebagai pengarah ke dokumentasi dependency utama.
